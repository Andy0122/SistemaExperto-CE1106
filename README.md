# OrientadorCE - Sistema Experto Vocacional

## Requisitos de Instalación
Para poder ejecutar este proyecto, necesitas tener instalado **SWI-Prolog** en tu computadora.

## Pasos para Ejecutar el Programa

1. Abre la consola de comandos (`cmd` o terminal) y navega hasta la carpeta del proyecto.
2. Escribe el siguiente comando para abrir el entorno de Prolog:
   swipl

3. Una vez adentro, carga el archivo lógico principal :
   consult('Logic.pl').

4. Inicia el programa ejecutando:
   iniciar.

5. El sistema hará preguntas en lenguaje natural sobre tus gustos e intereses.
   Responde con frases como "Me gustan las matemáticas" o "No mucho".

   Nota: si ves caracteres raros en la consola de Windows, cambia el código de página a UTF-8 antes de iniciar Prolog:
   chcp 65001

6. Para salir del bucle de prueba en cualquier momento, simplemente escribe la palabra:
   salir
